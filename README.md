## Лабораторная работа 8 - Принципы SOLID

### Задание
Провести рефакторинг классов репозитория в соответствии с 2-мя принципами SOLID и продемонстрировать работу полученных компонентов.

### Реализованные принципы SOLID

#### ISP (Принцип разделения интерфейсов)
- **`Printer`** - интерфейс только для печати
- **`Scanner`** - интерфейс только для сканирования
- **`Fax`** - интерфейс только для отправки факсов
- **`OldPrinter`** - класс, реализующий только необходимый функционал

**Преимущество:** Классы не зависят от методов, которые они не используют

#### DIP (Принцип инверсии зависимостей)
- **`MessageSender`** - абстракция для отправки сообщений
- **`EmailSender`** - реализация отправки email
- **`SmsSender`** - реализация отправки SMS
- **`NotificationService`** - сервис, зависящий от абстракции
