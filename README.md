## Лабораторная работа 4: Stream API, Gson, Lombok

### Выполненные задания:

1. **Список посетителей** - вывод всех посетителей и их количества
2. **Уникальные книги** - список всех книг без повторений
3. **Сортировка по году** - книги отсортированные по году издания
4. **Поиск по автору** - проверка наличия книг Jane Austen
5. **Максимум книг** - максимальное количество книг у одного посетителя
6. **SMS рассылка** - группировка пользователей по количеству книг

### Используемые технологии:
- Java Stream API
- Gson для работы с JSON
- Lombok для сокращения кода
- Git + GitHub Actions
